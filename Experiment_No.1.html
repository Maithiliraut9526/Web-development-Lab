Topic: Study of Web Technology Basics and Language Vectors

1. Core Web Technology Stack

HTML (Structure): The skeleton of the web. It uses tags to define headings, paragraphs, and links.
CSS (Presentation): The skin and clothing. It handles the layout, colors, and overall visual design.
JavaScript (Behavior): The muscles and brain. It adds interactivity, animations, and logic to the page.

2. Web Architecture & Protocols

Understanding how data travels between a user and a server:

  • HTTP/HTTPS: The rules for transferring data.
  • Client-Side: What the user sees in the browser (Frontend).
  • Server-Side: Where the data and logic live (Backend).

3. Comparative Study of Languages

Language Category Examples Role in Web Development
Markup Languages HTML, XML Defining and structuring data content.
Styling Languages CSS, SASS Applying visual rules and responsiveness.
Scripting Languages JavaScript, PHP, Python Creating dynamic content and database connections.
